# urler-Eb
Simple tool to grab all directories and urls of any web pages. built in python3.
# SUPPORTED PLATFORM
ANY-PLATFORM WITH PYTHON 3

# HOW TO USE
INSTALL FIRST

git clone https://github.com/josifkhan/urler-Eb

cd urler-Eb

pip install -r requirements.txt

python urler-eb.py


usages:
python3 urler-eb.py
